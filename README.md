# Event Countdown Manager

A lightweight countdown application built with HTML, CSS, and JavaScript. This project was created as a front-end exercise to practice JavaScript timers, DOM manipulation, and responsive UI design.

## Features

- Live countdown timer
- Create countdowns for upcoming events
- Dynamic countdown updates
- Responsive interface
- Simple and lightweight implementation

## Technologies

- HTML5
- CSS3
- JavaScript
